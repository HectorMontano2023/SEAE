# SEAE - Sistema de Evaluación de Alternativas Económicas

Proyecto desarrollado con Vaadin y Java para evaluar alternativas económicas.

## Funcionalidades
- Login de ejemplo
- Home de bienvenida
- Evaluación de dos alternativas
- Cálculo de Valor Presente
- Interfaz moderna.

## Tecnologías utilizadas
- Java
- Vaadin
- Maven
- Spring Boot

## Ejecución
```bash
mvn clean install
mvn spring-boot:run
